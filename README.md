# AndroidBasicProject
AndroidBasicProject是一个免费的、开源的、简易的Android基础项目，方便您快速的进行Android应用程序的开发。包含以下内容：
基础的Activity、Fragment；Event事件管理；下载管理；异常信息收集；日志打印；通用适配器；常用工具类。
